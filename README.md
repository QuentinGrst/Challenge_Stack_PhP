# Installation

# Base de données
Importer le fichier sell_me_out.sql dans votre base de données.
Remplir le fichier config.json avec les informations de votre base de données.

Exemple de fichier config.json:
{
    "database": {
        "database": "db",
        "host": "localhost",
        "user": "user",
        "password": ""
    }
}
